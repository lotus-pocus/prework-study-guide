HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The declaration !DOCTYPE html tells us to expect html on the page and is not case sensitive as it is a declaration
  • style defines the css styles associated with a website
  • base defines the base URL for a webpage
  • link connects an external resource to the HTML document
  • meta defines metadata e.g.character set, description, keywords, author, viewport
  • script embeds clientside scripts in the html document
  • div creates divisions in page
  • span creates an inline division
  • h1,h2,h3,h4,h5,h6 is the level of headings
  • p is paragraph or block of text
  • ul, ol, li means unorderede lists, ordered lists, list items
  • img is image
  • src defines the location of the image
  • br creaates a line of empty space or a line break between 2 blocks of content
  • a is anchor
  • href points to the URL for the link
  • alt attribute contains a string of text that describes the appearance and functionality of the image which helps improve search engine ranking

CSS

  • CSS is Cascading Style Sheets
  • CSS is the language used to describe the presentation of a website created with HTML

GIT

  • GIT is a software tool to verison our code
  • GIThub is cloud storage
  • BITbash is a unix based os
  • got status checks what branch we are currently on
  • git checkout -b branch-name:creates a new branch and switches to it
  • git add-A to commit all files in the working branch
  • git pull is used to fetch and download content from a remote repo and immediately update the locat repo
  • git clone to make a clone or copy of a repo in a new directory at another location. Copies an existing git repo
  • git commit captures a snapshot of the projects currently staged changes. It records changes in the repo
  • git push send the recent commit history to github
Javascript
  • Javascript is a lightweight programming language used to provide logic and interactivity for websites
  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • style.css is linked within the head element sript.js is linked within the body element
  • if statement is a way of introducing decision making into code
  • truthy is when the computer encounters an if statement in javascript, it will check to see if the condition that we provide is truthy.
  • Javascript reads from top to bottom
  • array is a group of related data
  • for loop is to perform a task on each piece of the data
  • quotation marks are only needed for strings. Numbers and Boolean values do not need quotation marks
  • Boolean value represents a truth value that is true or false
  • index is a unique number identifying location
  • 0 - Javascript arrays are zero indexed and sequential. The first item has an index of 0 not 1.
  • length is a built in property of an array, it sets or returns the number of elements in an array.
  • x++ is shorthand for x+1 for loop starts with for
  • A function is a set of instructions that tell the computer how to perform a certain task
